# UI 체크리스트 (필수 기능 검증)

이 체크리스트는 **필수 기능 UI 표기 검증**을 위해 사용한다.

---

## 1. 입력 영역
- [ ] 장바구니 아이템 추가/삭제가 동작한다.
- [ ] 배송비 입력이 가능하다.
- [ ] 가격 쿠폰 입력(%/정액)이 가능하다.
- [ ] 배송 쿠폰 입력(금액형)이 가능하다.

---

## 2. 결과 영역 (필수 표기)
- [ ] 최종 결제액이 표시된다.
- [ ] 쿠폰별 실제 할인액이 표시된다.
- [ ] 실질 할인율 2종(상품 기준/총액 기준)이 표시된다.
- [ ] minSpend 미달 시 **부족 금액**이 표시된다.
- [ ] 결과 영역에 **정책/기준(적용 순서, minSpend 기준, 반올림, clamping)**이 표시된다.

---

## 3. 추천 Top3
- [ ] Top3 결과가 표시된다.
- [ ] 정렬 기준(결제액 → 총할인액 → 입력 순서)이 일관되게 반영된다.
- [ ] 추천 사유(간단 이유)가 표시된다.

---

## 4. 예외/제약 표시
- [ ] 결제수단 제한 쿠폰이 비활성화되고 사유가 표시된다.
- [ ] minSpend 미달은 “적용 실패”로 표시된다.

---

## 5. 비교/대시보드(선택)
- [ ] 비교 화면(2개 선택)에서 결제액/할인율 비교가 가능하다.
- [ ] 대시보드 KPI 카드가 정상 표시된다.
